# Custom Channels Feature для Tonkeeper iOS

## Описание
Эта модификация добавляет возможность добавления пользовательских каналов в раздел "Коллекции" (Collectibles).

## Что добавлено:

### 1. Двойное нажатие на навигационную панель
- При двойном нажатии на заголовок "Коллекции" открывается диалоговое окно

### 2. Добавление пользовательских каналов
- В диалоге можно ввести username (например: `bobico`)
- После добавления создается карточка с изображением по URL:
  ```
  https://cache.tonapi.io/imgproxy/6bjc3arFMRcsqoh80US6jmQ6_OIflSLewvFd1XxcuIk/rs:fill:500:500:1/g:no/[base64_encoded_url].webp
  ```
  где base64_encoded_url = base64("https://nft.fragment.com/username/{username}.webp")

### 3. Локальное хранение
- Все добавленные каналы сохраняются в UserDefaults
- Каналы отображаются при каждом запуске приложения
- Каналы отображаются первыми в списке, затем идут NFT

## Измененные файлы:

1. **CollectiblesListViewController.swift**
   - Добавлен обработчик двойного нажатия
   - Добавлен диалог для ввода username
   - Обновлен dataSource для поддержки пользовательских каналов

2. **CollectiblesListViewModel.swift**
   - Добавлена структура CustomChannel
   - Добавлены методы для сохранения/загрузки каналов
   - Обновлена логика создания snapshot и models

3. **CollectiblesListItems.swift**
   - Добавлен новый тип элемента: customChannel

## Как использовать:

1. Откройте приложение Tonkeeper
2. Перейдите на вкладку "Коллекции" (Collectibles)
3. Дважды быстро нажмите на навигационную панель
4. Введите username (например: bobico)
5. Нажмите "Add"
6. Карточка канала появится в списке

## Технические детали:

- Порог для двойного нажатия: 0.5 секунды
- Хранилище: UserDefaults с ключом "customChannels"
- Формат изображений: WebP (500x500)
- API для изображений: cache.tonapi.io + imgproxy

## Сборка проекта:

```bash
# Установите зависимости
make setup

# Откройте проект в Xcode
open Tonkeeper.xcodeproj

# Выберите целевое устройство и соберите проект
```

## Примечания:

- Функция работает полностью локально
- Нет проверки существования username на сервере
- Изображения загружаются по требованию из кеша TON API
- Каналы не синхронизируются между устройствами
